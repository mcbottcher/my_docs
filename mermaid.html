Mermaid allows you to insert figures and diagrams into your markdown or re-structured text documents.

Usageïƒ

  • In Github markdown, you can use mermaid like you would a code block:

Example of Github Markdownïƒ
```mermaid

.. code goes here

```

  • In rst files, you can setup mermaid for you sphinx build with this: Python Package

Example using mermaid in sphinx rst fileïƒ
.. mermaid::

    graph TD
    A --> B
    B -- Label --> C
